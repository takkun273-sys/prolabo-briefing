# 申し送りシステム｜プロラボ加古川校

放課後等デイサービス プロラボ加古川校 スタッフ向け申し送り記録アプリです。

---

## ファイル構成

```
prolabo-kaikou/
├── index.html      メイン画面
├── style.css       スタイル
├── app.js          アプリロジック
├── sw.js           Service Worker（オフライン対応）
├── manifest.json   PWAマニフェスト
└── README.md       このファイル
```

---

## GitHub Pages へのデプロイ手順

1. GitHubで新しいリポジトリを作成（例：`prolabo-kaikou`）
2. このフォルダ内のファイルをすべてアップロード
3. リポジトリの「Settings」→「Pages」→ Branch: main / root → Save
4. 数分後に `https://takkun273-sys.github.io/prolabo-kaikou/` で公開されます

## 主な機能

- スタッフ別ログイン
- 本日の曜日に登録された児童のみ表示
- 体調チェック（良好／体調不良／気分ムラ／トラブルあり／保護者連絡要）
- 各項目への複数スタッフによる追記（職員名・時刻付き）
- 日付ごとの保存・履歴閲覧
- 管理者による曜日別児童設定
- PWA対応（ホーム画面追加・オフライン動作）

## カスタマイズ方法

### スタッフ・パスワードの変更
`app.js` の `DEFAULT_STAFF` 配列を編集してください。

```javascript
const DEFAULT_STAFF = [
  { name:"佐伯", password:"任意のパスワード", isAdmin:true },
  { name:"田中", password:"任意のパスワード", isAdmin:false },
  // 追加する場合はここに追記
];
```

### 初期の児童設定変更
`app.js` の `DEFAULT_CHILDREN` オブジェクトを編集してください。
（アプリ起動後は管理者設定画面からも変更可能です）

---

## 注意事項

- データは各端末のブラウザ（localStorage）に保存されます
- 複数端末でのデータ共有には別途バックエンド連携が必要です
- 児童の個人情報を扱うため、URLの共有範囲にご注意ください
